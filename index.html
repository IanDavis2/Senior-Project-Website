Web Design for the User

What is User Interface?

The user interface is the interaction between the computer and the user. Display screens and other features are considered to be a part of the user interface. It also refers to the way the user interacts with the computer, application, or website. The features which are interactive or infromative are generally included in the user interface.

What is User Experience?

The user experience is based around the needs of users of the product. In order for your website to have a good user experience, the information must fulfill a need or want, the site must be easy and convenient to use (usability), desirable, and accessable. These are the bases of a good user experience.
