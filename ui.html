User Interface

Navigational Components

Search Box: Also known as a search field, this is a box in which the user may enter words to get results back from the search engine or website.
Pagination: Pagination divides the content of the website into multiple pages (usually numbered) and allows the user to skip to any page by clicking
the box with the page's assigned number.
Icons: Icons are symbols of a particular page that allow users to easily recognize them and their association with the page should seem intuitive.
Tags: Tags show the user pages in the same category as what they have looked at, which allows them to more easily find pages relevant to them.
Track Bar: A track bar, or slider, allows the user to adjust a specific value using a scrolling point on a line, rather than typing in the value directly.
Image Carousel: An image carousel is a series of images users may scroll through and select, the images are often hyperlinked and lead to other pages.
Breadcrumbs: Breadcrumbs are a 'trail' of previous pages the user has visited before their current one, which links to the previous pages so that the
user can retrace their steps.
