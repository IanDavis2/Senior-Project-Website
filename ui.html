User Interface

Navigational Components

Search Box: Also known as a search field, this is a box in which the user may enter words to get results back from the search engine or website. Pagination: Pagination divides the content of the website into multiple pages (usually numbered) and allows the user to skip to any page by clicking the box with the page's assigned number. Icons: Icons are symbols of a particular page that allow users to easily recognize them and their association with the page should seem intuitive. Tags: Tags show the user pages in the same category as what they have looked at, which allows them to more easily find pages relevant to them. Track Bar: A track bar, or slider, allows the user to adjust a specific value using a scrolling point on a line, rather than typing in the value directly. Image Carousel: An image carousel is a series of images users may scroll through and select, the images are often hyperlinked and lead to other pages. Breadcrumbs: Breadcrumbs are a 'trail' of previous pages the user has visited before their current one, which links to the previous pages so that the user can retrace their steps.

Input Controls

Checkbox: A checkbox allows the user to choose options from a set. They should ususally be in a vertical list. Sometimes multiple columns are necessary Dropdown Lists: These allow users to choose one option at a time in a compact manner. Radio Buttons: These are used for users to choose one option at a time. List Boxes: Similar to dropdown lists, but allowing the user to choose multiple options. Buttons: A button has a label or symbol that indicates a particular action when it is clicked Dropdown Buttons: A dropdown button is a button that displays a dropdown menu when pressed. Toggles: A toggle allows the user to choose one of two binary options (eg. on and off), and it helps when these two options are visually distinct. Text Fields: Text fields allow the user to input text. Date and Time Pickers: A date or time picker allows the user to pick a date or time respectively, and the information entered is put into the system.

Informational Components

Notifications: A notification is a message sent to the user to inform them of something new to see. These are often used to announce the success or error at a task. Progress Bars: Progress bars tell the user how far they are through a series of stages in a process by filling in a bar on scree. Tool Tips: Tooltips let users see hints about an item when they hover their cursor over it. Message Boxes: Message boxes are windows that give the user information and make them take aan action before continuing. Modal Window (pop-up): Modal windows require users to fill in some information or interact witht he window before returning to the regular system.

Containers

Accordion: An accordion is a list of items that will show additional lists when an individual item is clicked or hovered over by the user.
